Способи функціонального застосування JavaScript

JavaScript -- мова програмування, яка дозволяє створити динамічно оновлюваний контент, управляє мультимедіа, анімує зображення, втім робить все, що завгодно.
Ядро мови JavaScript складається з деякої кількості звичайних можливостей, які дозволяють робити таке:
  • Зберігати дані всередині змінних. Наприклад, можна запитати користувача ввести власне ім'я, а потім зберегти ім'я у змінній name.
  • Операції над фрагментами текстів (відомими у програмуванні як "рядки"). Наприклад, беремо рядок "Player 1:" і приєднуємо його до значення змінної name для отримання повного тексту, наприклад: "Player 1: Chris".
  • Запускати код відповідно до певних подій, що відбуваються на веб-сторінці. Наприклад, можна використовувати подію click, для визначення моменту, коли кнопка була натиснута, відповідно до цього запускався код, який оновлював текст.
  • І багато іншого!
